## Setup

Run `npm install`

Before running anything locally, create a `.env` file and initialize your env variable like so:

    PORT=YOUR_PORT
    appId=YOUR_SHOPIFY_API_KEY
    appSecret=YOUR_SHOPIFY_SECRET_KEY
    appScope=write_products
    appDomain=YOUR_DOMAINE_NAME

Start dev server

    npm run dev

Start prod server

    npm start
